# Обзор Chart

## Есть Excel-файл, которое содержит определенные данные. Задача:
- Изобразить данные на 3D модели

## Использовались следующие ресурсы:
- PySide6 — Графический интерфейс приложения
- Openpyexcel — Библиотека для работы с Excel-таблицами
- OpenGL — Библиотека для изменения положения 3D модели

## Результат:
- Написано приложение на python
- В графическом интерфейсе есть необходимые инструменты для просмотра Excel-таблицы (старт, стоп, перемотка по строкам, регулировка скорости)
- Реализована возможность выбрать любую 3D модель
